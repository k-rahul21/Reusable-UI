Typography

Typography is used for styling the headings, paragraph of the web page.

Headings

There are 6 variants of headings available - h1 through h6. Use the tags h1 through h6 as per the preferred size.

Small text

We have 3 sizes available along with a grey variant. Use class p-lg for large text, p-md for medium text, p-sm for small text and text-grey for grey text. Adding no class results in the default text.
