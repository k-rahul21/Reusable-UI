Avatar

Avatar are typically used to display a user profile as a picture, an icon.

Image Avatar

You can create image avatar using the avatar class in img tag.

Sizes Avatar

You can change the size of avatar by adding the following classes avatar-sm , avatar-md , avatar-lg or avatar-xl.
