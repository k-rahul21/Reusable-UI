Button

Buttons allow user to take actions, and make choices with a single tap.

Basic Buttons

You can add the following class btn followed by the variant name like btn-primary , btn-success , danger-warning or btn-danger .

Outline Buttons

Outline buttons are created by using btn class, followed by outline-variant like btn-outline-primary , btn-outline-success , span> btn-outline-danger or btn-outline-warning .
