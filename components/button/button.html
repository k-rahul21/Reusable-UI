Button

Buttons are the soul of an interactive Web Application.Buttons allow user to take actions, and make choices with a single tap.

Basic Buttons

You can add the following class btn followed by the variant name like btn-primary , btn-success , danger-warning or btn-danger . 
                
                  <-- Using the btn class followed by the variant name -->

Outline Buttons

Outline buttons are created by using btn class, followed by outline-variant like btn-outline-primary , btn-outline-success , btn-outline-danger or btn-outline-warning . 
                
                  <-- Using the btn class followed by the variant name -->

Link Button

Want to attach a link action, at a button tap? Use Link Button.

Floating Action Buttons

Floating Action Button (FAB), shaped like a circled icon floating above the UI, is a tool that allows users to call out the key parts of your app.
